rocket_launch Getting Started

ZiframeBot adalah chatbot berbasis rule-based yang mudah diintegrasikan ke website Anda. Tidak memerlukan backend atau database, cukup tambahkan script dan konfigurasi rules dalam format JSON.

lightbulb Catatan: ZiframeBot bekerja sepenuhnya di client-side menggunakan JavaScript vanilla. Cocok untuk FAQ, customer support, dan lead generation.

power Integration Guide

Langkah 1: Tambahkan Script

Tambahkan script ZiframeBot ke dalam tag <head> atau sebelum closing tag </body>: 

<!-- Tailwind CSS (optional for styling) -->
<script src="https://cdn.tailwindcss.com"></script>

<!-- ZiframeBot CSS -->
<link rel="stylesheet" href="https://ziframebot.com/style.css">

<!-- ZiframeBot Script -->
<script src="https://ziframebot.com/script.js"></script>

Langkah 2: Tambahkan HTML Container

Buat container untuk chatbot di halaman Anda: 

<div id="chat-messages" class="chat-container"></div>
<textarea id="user-input" placeholder="Type your message..."></textarea>
<button id="send-button">Send</button>
<button id="clear-chat">Clear Chat</button>

Langkah 3: Konfigurasi Rules

Buat file rules.json di folder yang sama dengan script: 

{
  "rules": [
    {
      "keywords": ["halo", "hai", "hello"],
      "response": "Halo! Ada yang bisa saya bantu?"
    },
    {
      "keywords": ["produk", "layanan"],
      "response": "Kami menyediakan berbagai produk dan layanan..."
    }
  ],
  "fallback": "Maaf, saya belum memahami pertanyaan Anda."
}

settings Rules Configuration

Struktur JSON Rules

File rules.json memiliki struktur sebagai berikut:

rules (array)

Array yang berisi objek-objek rule. Setiap rule memiliki keywords dan response.

keywords (array of strings)

Kata kunci yang akan dicocokkan dengan input user. Matching bersifat case-insensitive dan menggunakan includes().

response (string)

Respons yang akan diberikan jika salah satu keyword cocok. Mendukung line breaks dengan \n.

fallback (string)

Respons default yang diberikan jika tidak ada keyword yang cocok.

Best Practices

  • check Gunakan variasi keyword untuk satu topik (contoh: "halo", "hai", "hello")
  • check Buat respons yang jelas dan informatif
  • check Urutkan rules dari yang paling spesifik ke yang paling umum
  • check Gunakan fallback response yang helpful dan mengarahkan user
  • check Test semua keyword secara berkala

sync How It Works

Berikut adalah alur kerja internal ZiframeBot:

1

User Input

User mengetik pesan dan menekan tombol kirim

2

Text Normalization

Input di-normalize (lowercase, trim whitespace)

3

Keyword Matching

Sistem mencocokkan input dengan keywords di rules.json

4

Response Generation

Jika match ditemukan, return response. Jika tidak, return fallback

5

Display & Save

Response ditampilkan di UI dan disimpan ke localStorage

User Input → Normalize → Match Keywords → Generate Response → Display
